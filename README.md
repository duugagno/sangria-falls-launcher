## Como usar

1. Baixe o instalador (`Sangria Falls Launcher-Setup-x.y.z.exe`) no link acima.
2. Execute e escolha seu idioma (Português, English, Español).
3. Abra o launcher — ele detecta o V Rising sozinho — e clique em **Instalar** nos itens da biblioteca.
4. **JOGAR.** 🩸

O launcher se atualiza sozinho: novas versões, mods e notícias chegam automaticamente.
